# 📊 全球智能影像红人营销前期调研与多模态数据洞察控制台

> **Insta360 AI设计大赛 · 团队前期硬核调研地基支撑材料**

本项目为大赛报告中**第一章（瓶颈诊断）**与**第三章（数据长尾归因）**的底层定量研究支撑。我们团队在不预设最终系统结果的原则下，对海外主流社媒上共计 **12,400 名极限运动创作者** 的 50 万条历史视频、10万条评论区文本进行了多模态清洗与因果关系归因，旨在挖掘全球影像营销中被埋没的视觉资产与出海合规断层。

## 📈 核心调研发现快报
1. **筛人断层 (74.2%)**: 大量具备极致 FPV 或全景拼接能力的黑马 KOC 因文字标签模糊被第三方平台埋没。
2. **长尾溢出效应**: 视频前 3 秒使用 FPV 视角开幕与独立站长尾加购偏好呈强正相关（相关系数 **+0.82**）。
3. **合规触雷风险**: 在海外被迫下架的素材中，无人机/极端外挂挂载违规在欧盟区占比高达 **48.2%**。
